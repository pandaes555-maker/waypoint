# Architectural Decisions

## 2026-07-04

### Decision

Use Vertical Slice Development.

### Reason

Every mission delivers one complete, working feature that can be tested, committed, and documented before moving to the next feature.

---

### Decision

Use Next.js App Router.

### Reason

Provides modern routing, server components, and scalability for the Waypoint MVP.

---

### Decision

Use Supabase for backend services.

### Reason

Provides authentication, PostgreSQL database, and storage with minimal infrastructure overhead.

---

### Decision

Repository documentation is the source of truth.

### Reason

Project continuity should depend on repository documentation rather than chat history.
