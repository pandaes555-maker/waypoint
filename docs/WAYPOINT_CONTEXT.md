# Project

Waypoint

---

# Vision

Waypoint is an AI-powered Progress Memory.

It helps people capture valuable knowledge from the internet and rediscover it exactly when they need it.

Waypoint is not a bookmark manager.

Waypoint is a personal AI Knowledge Layer.

---

# Core Promise

Save anything.

AI understands it.

Find it instantly later.

---

# Primary Users

* Founders
* Agency Owners
* Consultants
* Sales Professionals
* Students
* Knowledge Workers

---

# Core Problem

People save hundreds of useful links, videos and resources but never revisit them because traditional bookmarks and saved collections become unmanageable.

---

# Solution

One-click capture.

AI automatically understands the content.

Everything becomes searchable.

No folders.

No manual organization.

---

# Product Principles

Capture must take less than 2 seconds.

AI organizes everything.

Search is more valuable than folders.

Every saved item becomes a Knowledge Object.

Ship working software over perfect software.

---

# Knowledge Object

Each saved item contains:

* URL
* Title
* Description
* Domain
* Favicon
* Thumbnail
* Summary
* Knowledge Category
* Topics
* Keywords
* Reason To Revisit
* Favorite
* Notes
* Metadata
* Created Date
